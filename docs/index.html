What is it?

wkhtmltopdf and wkhtmltoimage convert HTML to PDF or images with Qt WebKit. They run headlessly and include a C library for embedding.

Get started

Use a prebuilt package from Downloads or build from source. Official packaging lives in wkhtmltopdf/packaging. 

wkhtmltopdf https://example.com example.pdf

Security

Do not render untrusted HTML. Read the project status and AppArmor example before using wkhtmltopdf with risky input.
