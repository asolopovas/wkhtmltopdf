What is it?

wkhtmltopdf and wkhtmltoimage convert HTML to PDF or images with Qt WebKit. They run headlessly and include a C library.

Install

Use a prebuilt package from Downloads or build from source. Packaging lives in wkhtmltopdf/packaging.

Use

wkhtmltopdf https://example.com example.pdf

See the generated CLI manual for options.

Library API

See the C library documentation.

Security

Do not render untrusted HTML. Read the project status and AppArmor example before using wkhtmltopdf with risky input.
