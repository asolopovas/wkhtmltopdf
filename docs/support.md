# Support

- Questions: [mailing list](http://groups.google.com/group/wkhtmltopdf-general) or [Stack Overflow](https://stackoverflow.com/questions/tagged/wkhtmltopdf).
- Bugs and feature requests: [GitHub issues](https://github.com/wkhtmltopdf/wkhtmltopdf/issues).
- Packaging: [wkhtmltopdf/packaging](https://github.com/wkhtmltopdf/packaging).
- Releases: [@wkhtmltopdf](https://twitter.com/wkhtmltopdf).

Bug reports need version, OS, exact command, minimal input, expected output, and actual output.
